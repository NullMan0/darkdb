# NullAPI

> Yeni Bir Npm Modülü!
 
## Fonksiyonlar!

- sifre : Rastgele Şifre Oluşturur
- kekoyazi : Girilen Yazıyı Keko Bir Şekilde Yazar
- tdk : Girilen Kelime TDK da varmı yokmu bakar

### Örnek ;
```js
const nullapi = require("null-api")

console.log(await nullapi.tdk("Elma"))
```
